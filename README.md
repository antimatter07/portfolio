# Matthew Villarica Portfolio

My portfolio website

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Structure

- `src/App.jsx`: portfolio content and sections
- `src/styles.css`: responsive page styling
- `public/_redirects`: Cloudflare Pages SPA fallback
- `vercel.json`: Vercel SPA fallback

## Deployment

### Vercel

Use the default Vite settings:

- Build command: `npm run build`
- Output directory: `dist`

### Cloudflare Pages

Use the same static build settings:

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Node version: `22`
